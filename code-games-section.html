Carrboro Hiking Trails App

This React project uses Google Maps API and REI's Hiking Project API to display a map of hiking trails around Carrboro, NC.

Try Demo
View Code

App Features:

  • clicking on map markers brings up trail details
  • user can filter trails by trail length
  • a service worker serves up assets quickly and enables offline mode
  • axios promise-based plugin for error handling
  • accessiblity standards met with ARIA attributes

MyReads App

This React project allows users to track books they are reading, have already read, and want to read. The user can search a limited API to select books to add to their bookshelves.

Try Demo
View Code

App Features:

  • search tool that takes user input and finds matches in a database of books
  • users can add books to shelves for Currently Reading, Want To Read, and Read
  • books can be moved between shelves or removed
  • react-router used for navigating between pages

Bug Dash! Arcade Game

Move your bug from the grass to the water using the arrow keys (or WASD). Collect the gems and watch out for boys!

Play Game
View Code

Game Features:

  • control the character using your keyboard
  • enemies have varying speed and starting points
  • player has 3 hearts that decrement with enemy encounters
  • collect gems to earn points
  • modal box shows game stats when the game is over
  • use the restart icon or replay button to start over
