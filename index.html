Bug Dash! Arcade Game

Move your bug from the grass to the water using the arrow keys (or WASD). Collect the gems and watch out for boys!

Play Demo
View Code

Game Features:

  • control the character using your keyboard
  • enemies have varying speed and starting points
  • player has 3 hearts that decrement with enemy encounters
  • collect gems to earn points
  • modal box shows game stats when the game is over
  • use the restart icon or replay button to start over

Scholarships

I was excited to be accepted into the Grow With Google Developer Scholarship in 2018. Our community of 25,000 aspiring developers learned HTML, CSS, and Javascript.

After this inital challenge period, 5,000 students were selected for the full Nanodegree scholarship in Front End Web Development. This course had more in-depth study of Javascript libraries including Ember, Angular, and React, and included accessibility best practices.

I was also lucky to be accepted into the Bertelsmann Data Science Scholarship in 2018. This community of 15,000 students took an Intro to Descriptive Statistics course where we studied data analysis, python, and SQL.

After completion I enrolled in the Business Analytics Nanodegree which built on these skills with projects featuring data visualization and data dashboards using Excel and Tableau.
